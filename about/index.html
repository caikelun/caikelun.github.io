About

Hi, my name is CAI Kelun (蔡克伦). I’m a software engineer living in China.

My primary area of expertise is Linux, Android, C, C++, ASM, network programming, crash analysis, reverse engineering. Also familiar with Java, Golang, Python, multimedia, database, distributed computing.

Works on Open Source Projects

  • xHook - a PLT hook library for Android native ELF.
  • libsvx - a socket network library for Linux.
  • PermissionBase - a permission framework using Nhibernate and Ajax for ASP.NET.
